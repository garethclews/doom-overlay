## Usage

``` sh
git clone https://github.com/hlissner/doom-emacs ~/.emacs.d
git clone https://gitlab.com/karetsu/doom-overlay ~/.config/doom
```

Edit `~/.config/doom/init.el` to your personal preference and then

```sh
cd ~/.emacs.d && make
```
